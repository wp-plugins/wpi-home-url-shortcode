== Installation ==

1. Upload folder `wpi-home-url-shortcode` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use Shortcode [wpi_home_url]

* Correct Shortcode entry: 
` [wpi_home_url]/products/`
` [wpi_home_url path=/products/]`
